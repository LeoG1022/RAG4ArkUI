# Skill Frontmatter Schema

> 每个 `.claude/skills/<name>.md`（AGENTS.md 除外）顶部必须含 YAML frontmatter，
> 让 skill 元数据**机器可读**。

---

## 完整字段示例

```yaml
---
name: generate                    # kebab-case，必须 = trigger 去 / 后
version: 1.0.0                    # semver；body 改动需 bump
trigger: /generate                # 用户输入命令，必须以 / 开头
description: 用一句话描述生成 KMP + ArkUI-X 双份最优实现   # 单行 ≤ 80 字
feature_log_required: true        # AI 驱动业务时是否必须留 feature log
classify_required: true           # 是否必须跑 classify-change.sh
preflight_required: true          # 是否必须跑 preflight.sh
calls:                            # 必跑的脚本（必须存在 + 可执行）
  - scripts/preflight.sh
  - scripts/check-api-parity.sh
  - scripts/classify-change.sh
  - scripts/new-feature.sh
references:                       # 必读的参考文件（必须存在）
  - .claude/references/arkuix-best-practices.md
---

# Skill: /generate
（正文不变）
```

---

## 字段说明

| 字段 | 类型 | 必填 | 校验 |
|---|---|---|---|
| `name` | string | ✅ | kebab-case；与文件名一致；与 trigger 去 / 后一致 |
| `version` | string | ✅ | semver（`X.Y.Z`） |
| `trigger` | string | ✅ | `/` 开头 |
| `description` | string | ✅ | 单行，非空，≤ 80 字符 |
| `feature_log_required` | bool | ✅ | true/false |
| `classify_required` | bool | ✅ | true/false |
| `preflight_required` | bool | ✅ | true/false |
| `calls` | list[string] | ✅ | 每个路径必须存在 + 可执行（路径相对 repo 根） |
| `references` | list[string] | ✅ | 每个路径必须存在 |

---

## 约定

- **正文不变**：frontmatter 只补充元数据，skill 的 prose 协议（Steps / Phase / Rules 等）保持不动
- **版本 bump 规则**：
  - skill 正文 / frontmatter 改动 → minor bump（`1.0.0` → `1.1.0`）
  - 协议性破坏（去掉 Step、改 inputs/outputs 语义）→ major bump（`1.x.x` → `2.0.0`）
  - 仅修 typo → 不 bump
- **豁免标记**：`feature_log_required: false` 替代旧的 `<!-- feature-log-exempt -->` HTML 注释（仅 `run-benchmark.md` 适用）

---

## 一致性规则（check-consistency.sh）

| 规则 ID | 检查 |
|---|---|
| `M-SKILL-FRONTMATTER` | 每个 skill 必含合法 frontmatter；必填字段齐全；name/trigger 一致；version 合法 semver |
| `M-SKILL-REF-VALID` | `calls` 路径必须存在且可执行；`references` 路径必须存在 |
| `M-SKILL-TABLE-SYNC` | `scripts/regenerate-skill-table.sh --check` 输出与 CLAUDE.md SKILL-TABLE 段一致 |

---

## 自动表格

`CLAUDE.md` Skill 速查段用注释标记包围：

```markdown
<!-- BEGIN SKILL-TABLE (auto-generated by scripts/regenerate-skill-table.sh, do not edit manually) -->
| Skill | 输入 | 输出 |
|---|---|---|
| `/generate` | 功能描述 | KMP + ArkUI-X 双份代码 |
...
<!-- END SKILL-TABLE -->
```

跑 `bash scripts/regenerate-skill-table.sh` 重写两标记间内容（幂等）。
